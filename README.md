# Deno Chatroom

Chat application developed using Deno, Typescript and Websockets

## Installation

Install [Deno](https://deno.land/#installation) and [Denon](https://deno.land/x/denon) 

## Usage

```node
$ denon run --allow-net --allow-read app.ts
```
